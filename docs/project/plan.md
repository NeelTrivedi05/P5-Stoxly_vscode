# Stoxly Implementation Plan

## Phase 0: Product Finalization (3-5 days)
1. Freeze MVP scope and acceptance criteria.
2. Finalize user input fields and recommendation outputs.
3. Confirm data source strategy for market/news signals.

Deliverables:
1. Final PRD.
2. Architecture v1.
3. UX wireframes including 3D scenes.

## Phase 1: Foundation Setup (1 week)
1. Bootstrap frontend (Next.js + TypeScript + Tailwind).
2. Set up backend API project and auth.
3. Set up database schema and migrations.
4. CI/CD baseline and environment configs.

Deliverables:
1. Running web app with login.
2. Base API and DB connected.

## Phase 2: Core User Flows (1-2 weeks)
1. Onboarding profile capture.
2. Watchlist CRUD.
3. Personalized news feed.
4. Alert settings and event pipeline v1.

Deliverables:
1. End-to-end onboarding to dashboard flow.
2. Seeded personalized feed.

## Phase 3: Intelligence Features (1-2 weeks)
1. Sentiment indicator service.
2. Trader activity spike detector.
3. Diversification recommendation rules engine.
4. Inflation-aware goal projection module.

Deliverables:
1. Sentiment and trader activity cards.
2. Diversification and projection dashboard.

## Phase 4: 3D Experience Layer (1 week)
1. Implement 3D portfolio scene.
2. Add 3D risk/horizon interaction.
3. Add fallback simple mode for low-end devices.
4. Performance optimization pass.

Deliverables:
1. Production-ready 3D dashboard mode.
2. Verified mobile fallback.

## Phase 5: Quality, Compliance, Launch (1 week)
1. Functional and integration testing.
2. Accessibility and performance audits.
3. Legal/compliance disclaimer pass.
4. MVP release checklist and launch.

Deliverables:
1. Launch candidate build.
2. Post-launch analytics dashboard.

## Suggested Backlog After MVP
1. Rebalancing automation suggestions.
2. Explainability deep panel for every recommendation.
3. Broker integration (read-only first).
4. Learning center for investor education.

## Team Shape (Lean)
1. 1 Product Manager.
2. 1 UI/UX Designer.
3. 2 Frontend Engineers.
4. 2 Backend Engineers.
5. 1 Data/ML Engineer.
6. 1 QA Engineer.

## Key Milestones
1. M1: Foundation complete.
2. M2: Core flows complete.
3. M3: Intelligence complete.
4. M4: 3D mode complete.
5. M5: Launch-ready MVP.
